# School-Supplies-Inventory-Sysytem
GROUP 3 - FINAL PROJECT
The Inventory Management System for School Supplies is a web-based system designed to efficiently manage and monitor school supply inventories. It replaces manual tracking with a digital system that allows users to add, update, view, and delete the item records easily.
The system keeps track of the item names, quantities, and prices, ensuring accurate and real-time inventory updates. It helps school staff easily monitor stock levels, prevent shortages, and improve organization and efficiency in managing supplies.
